# 🏠 家庭事项管理系统 (Family Management System)

一个基于 **Streamlit** + **AI** 构建的现代化、极简主义家庭事务管理平台，专为提升家庭协作效率而设计。

---

## 🌟 核心理念
本项目的核心目标是为家庭成员提供一个**清晰、直观、智能**的中心化管理界面，将繁杂的日常琐事与长期的日程规划有机结合。

## ✨ 主要功能

### 1. 📝 智能待办事宜 (To-Do List)
*   **AI 自然语言解析**: 利用 OpenAI GPT-4o 模型，自动理解用户输入的模糊时间。例如输入“周五下午五点拿快递”，系统会自动解析并提取为精确的日期。
*   **精准排序**: 任务根据解析出的截止时间（精确到分钟）进行自动排序，确保最重要的事永远排在最前面。
*   **极简审美**: 取消了冗余的边跨和阴影，采用连续的列表设计，视觉感官纯净、高效。
*   **状态追踪**: 点击任务左侧的选择框即可标记完成，已完成项会自动折叠/下沉。
*   **数据导出**: 支持一键导出当前的待办清单为 CSV 文件，采用 `utf-8-sig` 编码，完美适配 Excel。

### 2. 📅 家庭日历 (Google Calendar)
*   **双视图切换**: 采用标签页 (Tabs) 设计，可在“待办清单”与“日历”之间快速切换，默认优先显示待办事宜。
*   **日程模式 (Agenda)**: 日历默认以“日程”列表形式展示，方便快速查看近期的活动安排。
*   **多账号支持**: 默认绑定 `xuchunli@gmail.com`，支持在侧边栏动态配置其他 Google 日历 ID。

### 3. 🇸🇬 全系统新加坡时区对齐
*   无论是侧边栏的实时时钟显示，还是后端数据库的任务记录，全部基于 **新加坡 (SGT)** 时间运行。

### 4. 📱 优全平台访问优化
*   针对手机端进行了专门的 CSS 媒体查询优化，大尺寸触控目标和自适应布局，确保在手机浏览器上也能流畅使用。

### 5. 🔐 安全访问控制
*   **身份验证**: 全站受 6 位数字密码保护。
*   **缺省密码**: `790228`。
*   **安全退出**: 侧边栏提供“退出登录”按钮，确保家庭隐私。
*   **找回机制**: 登录页面显示开发者联系方式 `xuchunli@gmail.com`。

## 🔨 技术架构
*   **前端/应用框架**: Streamlit
*   **后端数据库**: SQLite (轻量级、无需配置)
*   **AI 引擎**: OpenAI API (GPT-4o)
*   **时区管理**: pytz
*   **数据处理**: Pandas

## 🚀 快速开始

### 1. 环境准备
确保已安装 Python 3.8+ 或 `uv` 工具。安装依赖：
```bash
uv add streamlit pandas openai python-dotenv pytz
```

### 2. 配置 AI Key
在根目录下新建 `.env` 文件，并填入您的 OpenAI API Secret Key：
```text
OPENAI_API_KEY=sk-xxxx...
```

### 3. 启动应用
在终端运行以下命令：
```bash
uv run streamlit run app.py
```
