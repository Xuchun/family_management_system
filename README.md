# 🏠 家庭管理系统 (Family Management System)

一个基于 **Streamlit** + **AI** 构建的现代化、极简主义家庭事务管理平台，专为提升家庭协作效率而设计。

---

## 🌟 核心理念
本项目的核心目标是为家庭成员提供一个**清晰、直观、智能**的中心化管理界面，将繁杂的日常琐事与长期的日程规划有机结合。

## ✨ 主要功能

### 1. 📝 智能待办事项 (To-Do List)
*   **AI 自然语言解析**: 利用 OpenAI GPT-4o 模型，自动理解用户输入的模糊时间。例如输入“周五下午五点拿快递”或“4月3日去旅游”，系统会自动解析并提取为精确的日期。
*   **多维度自动归类**: 系统通过智能判定，将任务切分为“⚡ 今日急需处理”、“🌙 明日事项”、“🗓️ 本周剩余事项” 及 “⏳ 本月剩余事项”等板块。
*   **精准排序**: 各区域内任务根据解析出的截止时间（精确到分钟）自动排序，确保最重要的事排在最前面。
*   **状态追踪与修改**: 支持直接划掉完成任务，并即时将其移入“已完成归档”区。支持随时点击“✏️”行内便捷修改任务内容。

### 2. � 高阶循环引擎 (Recurring Engine)
*   **智能频率提取**: 当用户输入“每周二买零食”时，AI 自动判定并锁定任务的循环特性。
*   **高级日界算法**: 原生支持并识别 `Monthly-LastDay`（每月最后一天）等极度复杂的跨月、跨平闰年的循环逻辑。
*   **自动投影衍生**: 系统能向未来整整一个月的时间线发起扫描，将长线的循环任务化为“虚拟分身”精确地显影在它应该出现的日历列表中。

### 3. 📊 所见即所得数据导出 (WYSIWYG Export)
*   **精准快照**: 提供一键下载 `.txt` 的功能。它不直接拖拽杂乱的底层数据，而是完整复制当前您在网页上看到的分类结构、任务顺序和排版，所见即所得。

### 4. 🇸🇬 全系统新加坡时区对齐
*   无论是侧边栏的实时时钟显示，还是后端数据库的任务记录，全部基于 **新加坡 (SGT)** 时间运行。

### 5. 📱 全平台界面优化
*   针对手机端进行了专门的 CSS 媒体查询优化，大尺寸触控目标和自适应布局，确保在手机浏览器上也能流畅使用。

### 6. 🔐 安全访问控制与持久登录
*   **物理级强力登录保护**: 针对各种复杂浏览器生态，引入了原生 Cookie 与 Javascript iframe 技术，强制维持您的登录会话。
*   **防误操作刷新**: 真正的防刷新掉线技术。登录状态受长达30天的持续保护。
*   **访问密码**: 请在 `.env` 文件或服务商的 Secrets 中设置 `APP_PASSWORD=你的密码`。

## 🔨 技术架构
*   **前端/应用框架**: Streamlit
*   **后端数据库**: SQLite (轻量级、无需配置)
*   **AI 引擎**: OpenAI API (GPT-4o)
*   **时区管理**: pytz
*   **数据处理**: Pandas

---

## 🚀 快速开始

### 1. 环境准备
确保已安装 Python 3.8+ 或 `uv` 工具。安装依赖：
```bash
uv add streamlit pandas openai python-dotenv pytz
```

### 2. 配置 AI Key
在根目录下新建 `.env` 文件，并填入您的 OpenAI API Secret Key：
```text
OPENAI_API_KEY=sk-xxxx...
```

### 3. 启动应用
在终端运行以下命令：
```bash
uv run streamlit run app.py
```
